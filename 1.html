Useful helpers

Here is the list of useful helpers. For more information please visit Assemble documentation.

Block helpers

  • {{#each object}} — iterate over specified object (private variables inside block: @index, @key)
  • {{#if condition}}
    {{#unless condition}}
    {{#is arg1 arg2}}
    {{#isnt arg1 arg2}}
    {{#gt arg1 arg2}}
    {{#gte arg1 arg2}}
    {{#lt arg1 arg2}}
    {{#lte arg1 arg2}}
    {{#compare leftValue operator rightValue}}
    {{#and arg1 arg2}}
    {{#or arg1 arg2}} — conditionally render a block (more helpers)
  • {{#markdown}} — render block as markdown

Inline helpers

  • {{default value defaultValue}} — provides a default or fallback value if a value doesn’t exist
  • {{formatDate date format}} — format date
  • {{now format}} — output current date and time
  • {{debug}} — log current context
